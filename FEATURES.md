# FEATURES

## Product Summary

ReplyJoy automatically creates draft replies for all emails in a user's Gmail Primary inbox. The SaaS is a lightweight place to view the latest drafts and manage reusable drafting rules that shape how those drafts are written.

## Product Rules

- The product auto-drafts for all Primary inbox emails.
- Users do not choose which emails get drafted.
- Drafts also live in Gmail.
- The SaaS does not manage inbox routing, sender rules, or approval workflows.
- The main setting the user can control is the drafting rules.

## Core User Flows

### 1. Connect Gmail

- User signs up and connects their Gmail account.
- Auto-drafting starts for Primary inbox emails.

### 2. View latest drafts

- User opens the SaaS and sees the most recent drafts.
- User can open a draft to read the email and the generated reply.
- User can continue working from Gmail if they want.

### 3. Configure drafting rules

- User adds or edits the drafting rules list.
- Future drafts use the updated rules.

## Features by Page

### Landing Page

- Simple explanation of the product: automatic draft replies for Gmail Primary inbox emails.

- Confirm that auto-drafting will run on all Primary inbox emails.

### Latest Drafts

- Show the newest drafts.

### Settings

- Save prompt changes for future drafts.
